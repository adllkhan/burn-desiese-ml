# Burn desiese ML Model — Диссертация AITU 2025

## Инструкция по компиляции в Overleaf

1. Загрузи этот ZIP через **New Project → Upload Project**
2. Зайди в **Menu** (верхний левый угол)
3. В разделе **Compiler** выбери **pdfLaTeX**
4. Нажми **Recompile**

## Структура проекта
- `main.tex` — основной файл (pdfLaTeX + T2A для кириллицы)
- `frontmatter/` — титул, аннотация, аббревиатуры
- `chapters/` — главы 1–5 + приложение
- `figures/` — 10 научных рисунков (PNG)
- `logos/` — логотип AITU
- `references.bib` — библиография (29 источников)
