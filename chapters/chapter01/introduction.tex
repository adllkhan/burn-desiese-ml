\section{Актуальность и мотивация исследования}

Ожоговые травмы представляют серьёзную глобальную проблему общественного здравоохранения: по данным Всемирной организации здравоохранения, ежегодно от ожогов страдают около 11~миллионов человек во всём мире, причём около 180~000 случаев заканчиваются летальным исходом~\cite{who2018burns}. Особенно высока распространённость ожоговых травм в странах с низким и средним уровнем дохода, где на них приходится значительная доля обращений в отделения неотложной помощи.

Ожоги классифицируются по глубине поражения тканей на три основные степени. Ожоги первой степени (поверхностные) затрагивают только эпидермис и характеризуются покраснением, болью и лёгким отёком. Ожоги второй степени (частичной толщины) проникают в дерму, вызывая образование волдырей и сильную боль. Ожоги третьей степени (полной толщины) разрушают все слои кожи, включая нервные окончания, и требуют хирургического вмешательства.

Точная и своевременная классификация степени ожога имеет критическое значение для выбора оптимальной тактики лечения: ожоги первой степени обычно лечатся амбулаторно, ожоги второй степени могут требовать госпитализации, а ожоги третьей степени нуждаются в интенсивной терапии и пересадке кожи. Однако даже опытные медицинские специалисты допускают ошибки при визуальной оценке глубины ожога, особенно в первые часы после травмы~\cite{monstrey2008assessment}.

Глубокое обучение и компьютерное зрение предлагают принципиально новый подход к диагностике: автоматический анализ изображений позволяет объективно оценивать степень ожога, снижая влияние субъективных факторов и ускоряя процесс принятия решений.

Казахстан, как страна со значительной промышленной активностью и развивающейся системой здравоохранения, имеет потребность в доступных и эффективных инструментах диагностики ожоговых травм. Внедрение ИИ-систем в клиническую практику может существенно повысить качество медицинской помощи, особенно в отдалённых регионах.

\section{Постановка задачи}

Клиническая проблема, рассмотренная в диссертации, — это автоматическая классификация степени ожогового поражения по фотографическим изображениям. Три ключевых ограничения мотивируют данное исследование:

\begin{enumerate}
    \item \textbf{Диагностическая сложность}: Визуальная оценка глубины ожога сильно зависит от опыта врача, времени, прошедшего с момента травмы, и условий освещения.
    \item \textbf{Доступность экспертизы}: Не все медицинские учреждения, особенно в сельской местности, имеют доступ к специалистам-комбустиологам.
    \item \textbf{Скорость диагностики}: В экстренных ситуациях важна быстрая предварительная оценка для сортировки пациентов и определения тактики лечения.
\end{enumerate}

Данная диссертация предлагает систему детекции объектов на основе архитектуры YOLOv8, которая адресует все три проблемы: обеспечивает объективную оценку по стандартной фотографии, не требует специального оборудования и работает в режиме реального времени.

\section{Цели и задачи исследования}

Основная цель — разработать, обучить и валидировать модель глубокого обучения для автоматической классификации степеней ожогов. Конкретные задачи:

\begin{enumerate}
    \item Провести систематический обзор существующих методов компьютерного зрения для анализа кожных поражений и ожогов.
    \item Подготовить и проанализировать набор данных Kaggle Skin Burn Dataset, охарактеризовав распределение классов и качество аннотаций.
    \item Реализовать конвейер предобработки данных с аугментацией для повышения обобщающей способности модели.
    \item Обучить модель YOLOv8n на подготовленном датасете с использованием трансферного обучения.
    \item Оценить производительность модели с использованием метрик mAP, precision и recall.
    \item Разработать REST API для практического применения модели в клинических условиях.
\end{enumerate}

\section{Научная новизна}

Научная новизна работы определяется четырьмя аспектами:

\textbf{1. Применение YOLOv8 для классификации ожогов.} Архитектура YOLOv8, современная система детекции объектов в реальном времени, применяется для задачи классификации ожоговых поражений. В отличие от традиционных CNN-классификаторов, YOLO позволяет не только классифицировать, но и локализовать область поражения на изображении.

\textbf{2. Формирование конвейера обработки медицинских изображений.} Разработан комплексный конвейер предобработки, включающий аугментацию данных (HSV-преобразования, масштабирование, зеркальное отражение), нормализацию и формирование YOLO-совместимых аннотаций.

\textbf{3. Метрики детекции объектов для медицинской диагностики.} Используется метрика mAP (mean Average Precision) с порогами IoU, адаптированная для оценки качества детекции ожоговых поражений:

\begin{equation}
    \text{mAP} = \frac{1}{|\mathcal{C}|} \sum_{c \in \mathcal{C}} \text{AP}_c
\label{eq:map}
\end{equation}

\noindent где $\mathcal{C}$ — множество классов (степеней ожогов); $\text{AP}_c$ — средняя точность для класса~$c$, вычисляемая как площадь под кривой precision-recall.

\textbf{4. Практическая система с REST API.} В отличие от чисто исследовательских работ, данная диссертация включает разработку готового к внедрению REST API на базе фреймворка Litestar, позволяющего интегрировать модель в существующие медицинские информационные системы.

\section{Практическая значимость}

Практические вклады данной работы:
\begin{itemize}
    \item Обученная модель YOLOv8n для классификации трёх степеней ожогов с весами, доступными для внедрения.
    \item Воспроизводимый конвейер обработки данных на Python (Ultralytics, PyTorch) с документированными модулями подготовки датасета, обучения и инференса.
    \item REST API для интеграции в клинические информационные системы, поддерживающий обработку JPEG-изображений и возвращающий структурированные результаты детекции.
    \item Методология обучения модели на облачных GPU-платформах (vast.ai), применимая для других задач медицинской визуализации.
    \item Готовое решение для внедрения в медицинские учреждения Казахстана в качестве вспомогательного инструмента диагностики.
\end{itemize}
