\section{Сводка вкладов}
В рамках данной диссертационной работы была разработана, обучена и валидирована система автоматической детекции и классификации ожоговых поражений, основанная на архитектуре глубокого обучения YOLOv8. Модель, обученная на датасете Kaggle Skin Burn Dataset, состоящем из $n{=}1\,225$ изображений с 3 классами ожогов, продемонстрировала mAP@0.5~=~0{,}847, precision~=~0{,}823, recall~=~0{,}791.

Основные научные и практические достижения:

\textbf{1.~Методологическое достижение.} Разработан воспроизводимый конвейер обработки медицинских изображений, состоящий из этапов загрузки датасета из Kaggle, преобразования в YOLO-совместимый формат, аугментации данных и обучения с трансферным обучением. Конвейер полностью автоматизирован и документирован.

\textbf{2.~Использование YOLOv8 для медицинской визуализации.} Впервые была применена современная архитектура YOLOv8 для задачи детекции ожоговых поражений. В отличие от традиционных классификационных подходов (CNN), детекция одновременно производит классификацию и локализацию области повреждения.

\textbf{3.~Практическая реализация.} Для интеграции модели с клиническими информационными системами был разработан REST API на основе фреймворка Litestar. API поддерживает обработку JPEG-изображений и возвращает структурированные в JSON формате результаты детекции.

\textbf{4.~Инференс в реальном времени.} Выбранная структура YOLOv8n демонстрирует скорость инференса $>$20 FPS даже на CPU, что позволяет использовать модель в условиях ограниченных вычислительных ресурсов.

\textbf{5.~Облачное обучение.} Отработана методология обучения глубоких моделей на облачной платформе vast.ai, предоставляющая доступ к GPU-ресурсам для студентов и исследователей.

\section{Научная новизна: итоги}

Сформулированные положения научной новизны (Глава~\ref{chap:intro}) были подтверждены:
\begin{itemize}
\item Архитектура YOLOv8, которая была задумана для решения общей задачи детекции объектов, успешно адаптирована для специализированной медицинской задачи --- классификация ожогов.
\item Используемая для оценки качества детекции ожоговых поражений формула mAP (уравнение~\ref{eq:map}) позволяет добиться объективного сопоставления с другими подходами.
\item Представлено продуманное гармоничное решение конвейера аугментации данных, состоящее из HSV-преобразований, геометрических искажений, Mosaic-аугментации, соответствующее требованиям медицинских изображений.
\end{itemize}

\section{Практические рекомендации}

На основании тех результатов, что мы получили в этой диссертации, сделаны следующие практические рекомендации решения основных задач:

\begin{enumerate}
\item \textbf{Сферы применения.} Модель идеально подходит для предварительного определения по величине степени ожога как вспомогательный инструмент к экспертному мнению врача-комбустиолога, но не для замещения его.

\item \textbf{Телемедицина.} Систему можно использовать для удалённой консультации, где пациент присылает фотографию ожога, а врач получает предварительную автоматическую оценку его тяжести.

\item \textbf{Сортировка пациентов.} В условиях массового поступления пострадавших (ЧС, катастрофы) автоматическая классификация позволяет своевременно определить приоритет лечения.

\item \textbf{Обучение персонала.} Модель в свою очередь может использоваться для обучения медицинских сестёр и фельдшеров распознаванию степеней ожогов.
\end{enumerate}

\section{Возможные направления будущих исследований}

Предлагаю несколько интересных направлений возможного дальнейшего расширения данной работы:

\begin{enumerate}
\item \textbf{Увеличение датасета.} Сбор и добавление новых изображений, особенно ожогов третьей степени, так как это позволит устранить дисбаланс классов. Рекомендованный минимальный объём - не менее 5\,000 изображений.

\item \textbf{Сегментация.} Переход от детекции (bounding box) к семантической сегментации позволит точнее определять площадь ожогового поражения, что критично для оценки TBSA (Total Body Surface Area).

\item \textbf{Временная динамика.} Анализ последовательности изображений одного и того же ожога в разные временные промежутки позволит достоверно предсказать как динамику прогрессирования, так и динамику заживления.

\item \textbf{Мультимодальность.} Интеграция дополнительных источников данных: термография, anamnesis (причина ожога, время получения травмы), демографические данные пациента.

\item \textbf{Мобильное приложение.} Поскольку нам требуется мобильное приложение для iOS/Android с встроенной моделью для offline-инференса на устройстве пациента, нам нужно будет его разработать.

\item \textbf{Клиническая валидация.} Исследование перспективы сопоставимого соответствия точности модели в сравнении с таковой эксперта-комбустиолога на выборке казахстанских пациентов.

\item \textbf{Интеграция с ИС.} Интеграция в виде REST API с медицинскими информационными системами казахстанских больниц для автоматической документации ожоговых травм.
\end{enumerate}

\section{Финальные замечания}

Использование глубокого обучения в классификации ожоговых повреждений представляет собой научно обоснованный и практически ценностный подход к усовершенствованию медицинской помощи при ожоговых травмах. В представленной диссертации показано, что современная архитектура детекции объектов YOLOv8, предварительно обученная на сравнительно небольшом специализированном датасете, способна достигать клинически приемлемой точности классификации степеней ожогов. Казахстанская система здравоохранения, активно перерабатывающая цифровые технологии, может значительно потянуть от внедрения таких технологий в экстренную и плановую медицинскую помощь. Созданная методикa предоставляет универсальный шаблон для переноса на иные задачи медицинской визуализации - классификации кожных заболеваний, оценке заживления ран и прочих дерматологических диагнозов. Абсолютно уместна предстает демонстрация возможности проведения исследований в области глубокого обучения с использованием облачных вычислительных мощностей (например, vast.ai), тем самым открывая доступ к современным методам ИИ для исследователей и студентов, которые еще не приобрели собственную GPU-инфраструктуру. Конечным мерилом успешности данной работы, является ее внедрение в клиническую практику - улучшение качества диагностики ожогов, сокращение времени принятия решения, и, как следствие, улучшение исходов лечения пациентов с ожоговыми травмами, как в Казахстане, так и за его пределами.
