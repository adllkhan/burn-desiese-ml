\chapter{Аннотация}
\begin{SingleSpace}

\noindent Ожоговые травмы представляют серьёзную глобальную проблему здравоохранения --- по данным Всемирной организации здравоохранения, ежегодно от ожогов страдают около 11 миллионов человек во всём мире. Точная и своевременная классификация степени ожога имеет критическое значение для выбора оптимальной тактики лечения: ожоги первой степени требуют амбулаторного лечения, тогда как ожоги третьей степени нуждаются в госпитализации и возможном хирургическом вмешательстве.

\vspace{0.8em}
\noindent В данной диссертации разработана система автоматической классификации степеней ожогов на основе глубокого обучения с использованием архитектуры YOLOv8. Модель обучена на наборе данных Kaggle Skin Burn Dataset, содержащем 1\,225 изображений ожоговых поражений кожи, распределённых по трём классам: ожоги первой степени (поверхностные), второй степени (частичной толщины) и третьей степени (полной толщины).

\vspace{0.8em}
\noindent Предложенная модель на основе YOLOv8n (nano) достигла точности детекции mAP@0.5 = 0.847, precision = 0.823, recall = 0.791. Архитектура YOLOv8 была выбрана благодаря её способности к детекции объектов в реальном времени и высокой эффективности при работе с медицинскими изображениями. Обучение модели проводилось на облачной GPU-платформе vast.ai с использованием трансферного обучения на предобученных весах COCO.

\vspace{0.8em}
\noindent Научная новизна заключается в применении современной архитектуры детекции объектов YOLOv8 для задачи классификации ожоговых поражений, формировании конвейера обработки медицинских изображений с аугментацией данных и разработке REST API для практического применения модели в клинических условиях.
