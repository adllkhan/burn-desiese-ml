\chapter{Аннотация}
\begin{SingleSpace}

\noindent Ожоговая травма является серьезной и глобальной проблемой здравоохранения --- по данным Всемирной организации здравоохранения, каждый год от ожогов страдает около 11 миллионов человек во всем мире. Правильная и своевременная классификация степени ожога является критической для оптимального выбора лечебной тактики: ожоги первой степени требуют амбулаторного лечения, в то время как ожоги третьей степени требуют госпитализации с возможным хирургическим вмешательством.

\vspace{0.8em}
\noindent В данной диссертации разработана система автоматической классификации степеней ожогов на основе глубокого обучения, использующая архитектуру YOLOv8. Модель обучалась на наборе данных Kaggle Skin Burn Dataset, содержащем 1\,225 изображений ожоговых поражений кожи, распределенных по трем классам: ожоги первой степени (поверхностные), второй степени (частичной толщины) и третьей степени (полной толщины).

\vspace{0.8em}
\noindent Предложенная модель на базе YOLOv8n (nano) потребила mAP@0.5 = 0.847, precision = 0.823, recall = 0.791. Архитектура YOLOv8 была выбрана благодаря способности детектировать объекты в реальном времени и высокой производительности для медицинских изображений. Обучение модели проводилось на облачной GPU-платформе vast.ai с использованием трансферного обучения на предобученные веса COCO.

\vspace{0.8em}
\noindent Научная новизна заключается в применении современного архитектуры детекции объектов YOLOv8 для задачи классификации ожоговых поражений, формировании конвейера для обработки медицинских изображений с аугментацией данных и разработке REST API для практического применения модели в клинических условиях.
